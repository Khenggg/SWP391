# ParkingBuilding.CoreApi

ASP.NET Core API skeleton.

Ownership:

- Auth/JWT
- User and driver write APIs
- Parking card, structure, entry/exit, payment, receipt, monthly pass, exception flows
- EF Core migrations and seed data
- `/api/core/*`

Code is intentionally not implemented yet. Students should add `.csproj`, `Program.cs`, controllers, services, entities, `ParkingDbContext`, migrations, and tests.
