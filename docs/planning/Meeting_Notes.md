# Meeting Notes

File này dùng để ghi lại quyết định quan trọng của team. Ghi theo thứ tự thời gian, mới nhất ở trên cùng hoặc dưới cùng nhưng phải thống nhất.

## Mẫu Ghi Biên Bản

```text
YYYY-MM-DD - Tên buổi họp

Người tham gia:
- ...

Nội dung chính:
- ...

Quyết định:
- ...

Action items:
- [ ] Việc cần làm - Người phụ trách - Deadline
```

## Nên Ghi Lại Khi

- Chốt sprint/task/P0.
- Đổi API contract, schema, ownership backend hoặc demo scope.
- Có blocker ảnh hưởng tiến độ.
- Có quyết định cắt hoặc dời feature.
