# Software Requirements Specification

File này là điểm vào cho tài liệu yêu cầu của dự án.

## Nguồn Đang Dùng

Tài liệu triển khai hiện tại cho kiến trúc 2 backend:

- `Developer_Implementation_Specification_Dual_Backend_NET_SpringBoot.md`

## Cách Sử Dụng

- Khi cần hiểu phạm vi, module, quyền sở hữu backend, API, bảng dữ liệu và test case, đọc tài liệu triển khai ở trên.
- Nếu sau này có bản SRS chính thức mới hơn, cập nhật link hoặc tóm tắt tại file này.
- Không ghi chi tiết sprint/task vào file này. Kế hoạch tiến độ nằm ở `../planning/Sprint_30_Day_Plan.md`.
